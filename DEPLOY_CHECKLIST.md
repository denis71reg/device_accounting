# Чеклист развертывания Device Accounting

## Перед развертыванием

- [ ] Создан файл `.env` на основе `env.example`
- [ ] Установлен уникальный `SECRET_KEY` (сгенерирован новый)
- [ ] Проверен `DATABASE_URL` (по умолчанию SQLite)
- [ ] Проверен `APP_PORT` (по умолчанию 5001)
- [ ] Установлены Docker и Docker Compose
- [ ] Проверено, что порт 5001 свободен

## Развертывание

- [ ] Выполнен `./deploy.sh` или команды вручную
- [ ] Приложение запущено: `docker-compose -f docker-compose.prod.yml up -d`
- [ ] Применены миграции: `docker-compose -f docker-compose.prod.yml exec app flask db upgrade`
- [ ] Создан первый супер-администратор
- [ ] Проверен доступ к приложению: `http://server-ip:5001`

## После развертывания

- [ ] Проверена регистрация нового пользователя
- [ ] Проверен вход в систему
- [ ] Проверены права доступа (супер-админ, админ, пользователь)
- [ ] Проверено создание/редактирование сотрудников
- [ ] Проверено создание/редактирование девайсов
- [ ] Проверена работа истории изменений (для супер-админа)

## Настройка Nginx (опционально)

- [ ] Установлен Nginx
- [ ] Создана конфигурация на основе `nginx.conf.example`
- [ ] Проверена конфигурация: `nginx -t`
- [ ] Активирован сайт
- [ ] Перезагружен Nginx: `systemctl reload nginx`
- [ ] Проверен доступ через домен

## SSL/HTTPS (опционально)

- [ ] Установлен Certbot
- [ ] Получен SSL сертификат: `certbot --nginx -d your-domain.com`
- [ ] Настроен автоматический обновление сертификата
- [ ] Проверен доступ по HTTPS
- [ ] Проверен редирект с HTTP на HTTPS

## Безопасность

- [ ] Изменен дефолтный `SECRET_KEY`
- [ ] Ограничен доступ к порту 5001 (только через Nginx)
- [ ] Настроен файрвол (если используется)
- [ ] Настроены регулярные резервные копии БД
- [ ] Проверены права доступа к файлам

## Мониторинг

- [ ] Настроен просмотр логов: `docker-compose -f docker-compose.prod.yml logs -f`
- [ ] Проверено использование ресурсов: `docker stats`
- [ ] Настроены уведомления об ошибках (опционально)

## Резервное копирование

- [ ] Настроен автоматический бэкап базы данных
- [ ] Проверено восстановление из бэкапа
- [ ] Документирован процесс восстановления

## Документация

- [ ] Обновлена документация для команды
- [ ] Задокументированы учетные данные (в безопасном месте)
- [ ] Создана инструкция для новых администраторов
